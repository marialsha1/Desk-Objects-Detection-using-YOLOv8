# Desk-Objects-Detection-using-YOLOv8

This is a simple computer vision project where I used YOLOv8 to detect objects in images.

## 🔧 Tools Used
- Python
- OpenCV
- Ultralytics YOLOv8
- Matplotlib

## 📁 Project Structure
- images/: input images
- runs/: output results with bounding boxes
- project.ipynb: main code

## 🚀 What I did
- Loaded a pretrained YOLOv8 model
- Ran object detection on custom images
- Displayed and saved results
- Filtered detections based on confidence
